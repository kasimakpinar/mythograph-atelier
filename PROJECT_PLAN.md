# Project Plan

The full planning document is kept in the repository root as `PROJECT_PLAN_Mythograph_Atelier.md`.

It is intentionally ignored by Git because it is a working planning artifact rather than part of the shipped Space.

This repository starts with the MVP implementation path:

1. Scaffold a working Gradio app.
2. Add controlled dynamic UI actions.
3. Track hidden objectives in session state.
4. Generate a validated art recipe.
5. Render an abstract fallback artwork locally.
6. Show the final gallery card with title, explanation, and symbol map.
7. Add model integrations for llama.cpp and a small image model.
8. Add trace sharing, fine-tuning data, and final submission polish.
